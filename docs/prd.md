# ServicePratic - Product Requirements Document

## Visão Geral

ServicePratic é uma aplicação web destinada ao gerenciamento de ordens de serviço para oficinas mecânicas, assistências técnicas e prestadores de serviços.

O sistema centraliza informações de clientes, equipamentos e atendimentos, proporcionando maior organização e rastreabilidade dos serviços realizados.

---

## Problema

Muitas empresas utilizam papel, planilhas ou aplicativos de mensagens para controlar atendimentos.

Isso gera:

- Perda de informações
- Falta de histórico
- Dificuldade de acompanhamento
- Problemas na geração de relatórios

---

## Objetivo

Desenvolver uma aplicação web para:

- Gerenciar clientes
- Gerenciar equipamentos
- Controlar ordens de serviço
- Registrar histórico
- Gerar indicadores operacionais

---

## Público-Alvo

- Oficinas mecânicas
- Assistências técnicas
- Empresas de manutenção
- Técnicos autônomos

---

## Funcionalidades MVP

- Login
- Cadastro de clientes
- Cadastro de equipamentos
- Abertura de OS
- Alteração de status
- Histórico
- Pesquisa de clientes
- Pesquisa de OS
- Relatórios
- Dashboard
